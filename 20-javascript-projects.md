# 20 Javascript projects

1. **Number Guessing Game (Easy)**
   - Create a program that picks a random number between 1 and 100. Let the user guess the number and provide hints ("Too high" or "Too low") until they guess it correctly.

2. **Simple Calculator (Easy)**
   - Build a basic calculator that can add, subtract, multiply, and divide two numbers entered by the user.

3. **To-Do List App (Easy)**
   - Develop a simple to-do list where users can add, view, and remove tasks in the browser using prompt/alert or DOM manipulation.

4. **Palindrome Checker (Easy)**
   - Take a string input and check if it reads the same forwards and backwards. Display the result to the user.

5. **Countdown Timer (Easy)**
   - Let the user enter a number of seconds and then display a countdown in the browser or console, updating every second until zero.

6. **Dice Rolling Simulator (Easy/Medium)**
   - Simulate rolling one or more dice and display the result. Allow the user to choose how many dice to roll.

7. **Password Generator (Medium)**
   - Create a tool that generates a random password containing letters, numbers, and symbols. Let the user select the password length.

8. **Contact Book (Medium)**
   - Build a mini-app where users can add, search, and remove contacts (name, phone, email) stored in an array or localStorage.

9. **Quiz Game (Medium)**
   - Develop a multiple-choice quiz with a set of questions. Track the score and show the final results.

10. **Currency Converter (Medium)**
    - Write a program that converts between currencies using exchange rates from a public API.

11. **Weather App (Medium)**
    - Fetch and display the current weather for a given city using a weather API (e.g., OpenWeatherMap).

12. **Hangman Game (Medium)**
    - Implement the classic Hangman game in the browser or console, letting users guess letters of a hidden word.

13. **Tic-Tac-Toe Game (Medium)**
    - Build a two-player Tic-Tac-Toe game playable in the browser using the DOM to display the board.

14. **File Organizer Tool (Medium/Hard)**
    - Use Node.js to create a script that organizes files in a directory by their type (e.g., images, PDFs, documents).

15. **Web Scraper (Medium/Hard)**
    - With Node.js and a library like Axios or Cheerio, scrape a website for data (like headlines or quotes) and print/save the results.

16. **Markdown to HTML Converter (Hard)**
    - Write a program that takes a Markdown file and converts it to basic HTML, either in the browser or with Node.js.

17. **Expense Tracker (Hard)**
    - Build an app to record daily expenses and generate reports/charts. Store data locally in the browser.

18. **Chat Application (Hard)**
    - Create a simple web-based chat using WebSockets (with Node.js and socket.io) for real-time messaging.

19. **Blog Website (Hard)**
    - Build a simple blog with JavaScript, allowing users to create, edit, and delete posts. Use localStorage for data persistence.

20. **Personal Portfolio Website with Admin Panel (Hard)**
    - Develop a portfolio website to showcase your projects and skills. Build an admin section for editing content dynamically and store data locally or via a Node.js backend.
