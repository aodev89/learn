# 20 Python projects from easy to hard

1. **Number Guessing Game (Easy)**
   - Write a Python script that generates a random number between 1 and 100 and asks the user to guess it. Give hints ("Too high/Too low") until the user guesses correctly.

2. **Simple Calculator (Easy)**
   - Build a calculator that can add, subtract, multiply, and divide two numbers entered by the user.

3. **To-Do List App (Easy)**
   - Create a command-line to-do list where users can add tasks, view all tasks, and mark tasks as done.

4. **Palindrome Checker (Easy)**
   - Take input from the user and check if the provided string is a palindrome (reads the same forwards and backwards).

5. **Countdown Timer (Easy)**
   - Let the user enter a number of seconds, then count down to zero, printing the time remaining each second.

6. **Dice Rolling Simulator (Easy/Medium)**
   - Simulate rolling one or more dice and display the result. Let the user decide how many dice to roll.

7. **Password Generator (Medium)**
   - Generate a random password with letters, numbers, and symbols. Allow the user to choose the password length.

8. **Contact Book (Medium)**
   - Build a command-line program to save, search, and manage contacts (names, phone numbers, emails) using a dictionary.

9. **Quiz Game (Medium)**
   - Create a quiz that asks multiple-choice questions from a list, checks answers, and shows the final score.

10. **Currency Converter (Medium)**
    - Write a program that converts between currencies using real-time exchange rates (using an API like exchangerate-api).

11. **Weather App (Medium)**
    - Fetch and display the current weather for a given city using an API (e.g., OpenWeatherMap).

12. **Hangman Game (Medium)**
    - Implement the Hangman word guessing game for the terminal.

13. **Tic-Tac-Toe Game (Medium)**
    - Create a two-player Tic-Tac-Toe game that runs in the terminal.

14. **File Organizer (Medium/Hard)**
    - Write a script to organize files in a folder by their type (e.g., move all images to an "Images" folder).

15. **Web Scraper (Medium/Hard)**
    - Scrape a website for specific information (like quotes, headlines, or prices) and save the data to a file.

16. **Markdown to HTML Converter (Hard)**
    - Write a program that reads a Markdown (.md) file and converts it into basic HTML.

17. **Expense Tracker (Hard)**
    - Build a CLI tool or simple GUI app to record daily expenses and generate summary reports.

18. **Chat Application (Hard)**
    - Create a simple chat application in the terminal using sockets to allow messaging between computers on the same network.

19. **Blog Website with Flask/Django (Hard)**
    - Design a simple blog with user authentication, the ability to create and edit posts, and comments, using a web framework like Flask or Django.

20. **Personal Portfolio Website with Database (Hard)**
    - Develop a web application that displays your skills, projects, and contact information. Allow dynamic editing of content via an admin panel and store data in a SQLite or PostgreSQL database.
